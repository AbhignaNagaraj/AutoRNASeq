# AutoRNASeq

Automated RNA-seq analysis pipeline using:

- Nextflow
- Docker
- Python
- Streamlit

## Features

- RNA-seq workflow automation
- Quality control
- Alignment
- Quantification
- Differential expression analysis
- Interactive dashboard

## Workflow

FASTQ → QC → Alignment → Quantification → Differential Expression → Dashboard

## Tech Stack

- Nextflow
- Docker
- Python
- Streamlit
- HISAT2
- FastQC
- featureCounts
- DESeq2
